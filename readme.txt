Professional Hebrew budget management system for WordPress with private budgets, expense tracking, and collaborative features.

== Description ==

Budgex is a comprehensive Hebrew budget management plugin for WordPress that allows registered users to create and manage private budgets with advanced collaboration features.

= Key Features =

* **Private Budget Management** - Each user can create multiple private budgets
* **Hebrew Language Support** - Full RTL support with Hebrew translations
* **Collaborative Budgeting** - Invite other users as viewers or administrators
* **Real-time Calculations** - Automatic budget tracking and remaining amount calculations
* **Professional Dark Mode** - Modern, eye-friendly interface
* **Mobile Responsive** - Works perfectly on all devices
* **Secure & Private** - Advanced permission system with role-based access

= Main Functionality =

* Create unlimited budgets with monthly limits
* Add, edit, and delete expenses (outcomes)
* Real-time budget calculations
* Invite users via email with specific permissions
* Dashboard with budget overview and statistics
* Export and manage budget data
* Professional Hebrew interface

= Perfect For =

* Personal finance management
* Family budget planning
* Small business expense tracking
* Team budget collaboration
* Hebrew-speaking users

== Installation ==

1. Upload the `budgex` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Ensure user registration is enabled on your site
4. Users can access budgets via the dashboard at `/budgex-dashboard/`

== Frequently Asked Questions ==

= Do users need to register to use Budgex? =

Yes, Budgex requires user registration and login to create and manage budgets for privacy and security.

= Can I share my budget with others? =

Yes, you can invite other registered users to view or help manage your budgets with different permission levels.

= Is the plugin available in Hebrew? =

Yes, Budgex is fully translated to Hebrew with complete RTL support.

= Does it work on mobile devices? =

Yes, Budgex features a fully responsive design that works perfectly on smartphones and tablets.

= Can I export my budget data? =

Budget data is stored in your WordPress database and can be accessed through the dashboard interface.

== Privacy Policy ==

Budgex stores all budget data locally in your WordPress database. No data is transmitted to external services. User budgets are private by default and only accessible to invited users with proper permissions.
